# Changelog

## [1.0.0] - 2026-02-25

### Initial Release

Production-ready SOCKS5 UDP checker with support for both IP-authenticated and username/password-authenticated proxies.

**Features:**
- IP authentication support
- Username/password authentication
- Real UDP traffic testing
- Clean CLI interface with argparse
- Quiet mode for scripting
- Batch testing script included
- Cross-platform support
- Zero dependencies

**Files:**
- `udp-check.py` - Main UDP checking tool
- `batch-test.sh` - Batch testing script for proxy lists
- `README.md` - Complete documentation

- `CHANGELOG.md` - This file
